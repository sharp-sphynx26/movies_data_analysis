
#  Movie Market Analysis Project

##  Project Overview
This project analyzes movie trends to understand what types of movies are popular in today's market and what modern audiences prefer.

The analysis includes data cleaning, exploratory data analysis (EDA), visualization, and SQL-based querying for deeper insights.

----

##  Problem Statement
To analyze movie data and identify:
- Most popular genres
- Relationship between genres and vote averages
- Market trends in movie consumption
- Insights that can help understand audience preferences

-------

##  Dataset Information
The dataset contains the following columns:

- `title` – Movie name
- `genre` – Category of the movie
- `vote_average` – Average rating given by users
- `overview` – Short description of the movie
