
# Movie Market Analysis Project

## Project Overview

This project analyzes movie industry trends to understand what types of movies are popular in today's market and what modern audiences prefer.

The project demonstrates an end-to-end data analysis workflow including data cleaning, exploratory data analysis (EDA), visualization, and SQL-based analysis.

---

## Problem Statement

The goal of this project is to analyze movie data to:

- Identify the most popular genres
- Analyze vote average distribution
- Understand audience preferences
- Extract meaningful insights using Python and SQL

---

## Dataset Description

The dataset contains the following columns:

- title – Movie name  
- genre – Category of the movie  
- vote_average – Average user rating  
- overview – Short description  

The raw dataset was cleaned and prepared for analysis before performing EDA and SQL operations.

---

## Tools & Technologies Used

- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook
- MySQL
- Git & GitHub

---

## Data Cleaning Steps

- Removed duplicate records  
- Handled missing values  
- Corrected data types  
- Applied Z-score method to handle outliers  
- Prepared dataset for SQL database import  

---

## Exploratory Data Analysis

- Analyzed genre distribution  
- Studied vote average patterns  
- Visualized rating distributions  
- Identified trends in audience preferences  

All analysis is documented in the Jupyter Notebook file.

---

## SQL Analysis

After cleaning the data:

- Imported the dataset into MySQL  
- Performed aggregation queries  
- Used GROUP BY, ORDER BY, filtering, and ranking queries  
- Extracted analytical insights from structured data  

---

## Key Insights

- Certain genres dominate the market.
- Higher-rated movies tend to cluster in specific categories.
- Audience preference patterns can help guide content strategy.
